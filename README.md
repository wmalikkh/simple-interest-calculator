# Simple Interest Calculator

A shell script that calculates simple interest based on principal, rate, and time inputs.

## Description
This project was created as part of the Introduction to Git and GitHub course. It contains a bash script, `simple-interest.sh`, that takes principal, rate, and time as inputs and calculates the simple interest.

## Usage
Run the script from the command line:
```bash
./simple-interest.sh
```
